# CalculadoraSimples 

Uma calculadora de linha de comando bem simples, mas com tratamento de erros decente!

## Oq é isso?

É um programa em Java que faz operações matemáticas básicas:
- Soma (+)
- Subtração (-)
- Multiplicação (*)
- Divisão (/)

Nada de complicado, só o essencial mesmo.

## Como usar

1. **Compilar:**
```bash
javac CalculadoraSimples.java
